How it works

Apple-first foundations keep data secure and enforcement reliable.

Every step is designed around Screen Time entitlements so families can trust what they see and how limits apply across devices.

1. Authorize Screen Time

SocialScreen requests the Screen Time Communication entitlement and walks users through granting permission on their devices.

2. Select activities

FamilyActivityPicker captures the apps, sites, and categories to manage, storing the encrypted list locally.

3. Apply policies

ManagedSettingsStore enforces shields while DeviceActivitySchedule automates when limits turn on or off.

4. Review insights

The dashboard refreshes with live usage metrics so families can review progress during check-ins.

Why build it this way

First-party APIs, SwiftUI velocity, backend readiness.

Apple's Screen Time frameworks deliver reliable enforcement, while SwiftUI makes it easy to share UI between iPhone and future macOS/iPad builds. A dedicated backend will power private sharing without sacrificing control.
