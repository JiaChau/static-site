My Git-Hosted Website

A simple site ready for Git & GitHub Pages

About This Site

This is a minimal static website you can host from a Git repository, for example on GitHub Pages, GitLab Pages, or another static host.

How to Edit

  • Edit this text in index.html.
  • Change colors and layout in styles.css.
  • Add interactivity in script.js.

Next Steps

  1. Initialize a Git repository here.
  2. Push it to your Git hosting (e.g., GitHub).
  3. Enable Pages hosting in the repo settings.
